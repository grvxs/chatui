# chatui

Simple chat application.

## Installation

Clone the project:

```bash
git clone https://github.com/grvxs/chatui.git
```

Move to the project directory:

```bash
cd chatui
```

Run the server:

```bash
cargo run --release --bin chatui_server
```

Run the client:

```bash
cargo run --release --bin chatui_client
```

You can connect multiple clients.
